Install
=======
Put this in your .bashrc::
  source svn-ignore.bash

Restart your bash (or reload your .bashrc).

Usage
=====
Example 1::

  $ cd your-svn-working-copy
  $ svn-ignore-add file-you-want-ignore

Example 2::

  $ svn-ignore-add your-svn-working-copy/file-you-want-ignore
